# 🚀 Algorithm Explorer

A dynamic, interactive web application designed to visualize and compare the efficiency of different algorithms. This project demonstrates the practical implementation of search and sort functionalities within a user-friendly interface.

## 🌐 Live Demo
Check out the live application here: [Algorithm Explorer](https://abbasmalikzada.github.io/AlgorithmExplorer/)

## ✨ Features
* **Interactive Visualization:** Watch how different algorithms process data in real-time.
* **Performance Comparison:** Compare time complexities and execution speeds.
* **Clean UI:** Built with modern CSS for a responsive and intuitive user experience.
* **Phonebook Integration:** Includes a specialized module for exploring phonebook dataset searching.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Data Processing:** Python (Scripts included for dataset management)
* **Hosting:** GitHub Pages

## 📂 Project Structure
* `index.html` - The main entry point for the web interface.
* `script.js` - Contains the core logic for algorithm animations and calculations.
* `styles.css` - Custom styling for the application.
* `PhoneBook.py` - Python script for handling larger phonebook datasets.
* `ALGORITHM_COMPARISON.md` - Detailed breakdown of the algorithm analysis.
